# Track 2: Prompt Engineering (2 weeks)

## Outcomes
- Write prompts that are specific, testable, and reusable.
- Use role, constraints, examples, and output schema effectively.
- Evaluate prompt quality with repeatable criteria.

## Modules

### Module 1 — Prompt Anatomy
- Intent
- Context
- Constraints
- Output format

**Lab:** Rewrite weak prompts into structured prompts.

### Module 2 — Prompt Patterns
- Zero-shot and few-shot prompting
- Step-by-step decomposition
- Rubric-guided responses
- Self-critique and revision loops

**Lab:** Create a prompt library for 3 job scenarios.

### Module 3 — Structured Outputs
- JSON schemas
- Field validation
- Error handling for malformed output

**Lab:** Build a content-brief generator returning valid JSON.

### Module 4 — Prompt Evaluation
- Accuracy
- Consistency
- Brevity vs completeness
- Cost and latency awareness

**Lab:** Run 20-case test set across 2 prompt variants.

## Assessment
- Prompt pack submission (10 reusable prompts)
- Evaluation report with pass/fail thresholds
