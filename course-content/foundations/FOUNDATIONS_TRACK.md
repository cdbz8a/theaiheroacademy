# Track 1: AI Foundations (2 weeks)

## Audience
Absolute beginners and career switchers.

## Outcomes
- Explain the differences between AI, ML, and LLMs.
- Identify useful and risky AI use cases.
- Understand tokens, context windows, and model limitations.

## Module Plan

### Module 1 — What AI Is (and Is Not)
**Objectives**
- Distinguish rule-based systems from machine learning systems.
- Describe where LLMs are strong and weak.

**Activity**
- Classify 10 everyday tools by AI type.

**Artifact**
- One-page “AI opportunities in my job” brief.

---

### Module 2 — How LLMs Work (Practical View)
**Objectives**
- Explain next-token prediction in plain language.
- Describe effects of prompt context and examples.

**Activity**
- Compare outputs with short vs rich context prompts.

**Artifact**
- Prompt improvement notes with before/after outputs.

---

### Module 3 — Reliability Basics
**Objectives**
- Spot hallucinations and low-confidence output.
- Add lightweight verification checks.

**Activity**
- Fact-check an AI-generated summary against source text.

**Artifact**
- Reliability checklist v1.

---

### Module 4 — First Workflow
**Objectives**
- Chain input → model → review → output.
- Capture simple success metrics.

**Activity**
- Build a meeting-summary workflow.

**Artifact**
- Demo link + workflow diagram.

## Assessment
- Quiz (15 questions)
- “Explain AI to a non-technical stakeholder” video (2 minutes)
