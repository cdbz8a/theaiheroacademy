# Track 3: Build with APIs (3 weeks)

## Outcomes
- Integrate an LLM API into an app.
- Implement backend prompt orchestration.
- Add caching, retries, and guardrails.

## Modules

### Module 1 — API Basics
- API keys and environment variables
- Request/response lifecycle
- Basic error handling

**Lab:** CLI app that summarizes text input.

### Module 2 — Backend Workflow Design
- System prompts and task prompts
- Function/tool calls
- Context retrieval patterns

**Lab:** Build a FAQ assistant with source snippets.

### Module 3 — Productization
- Usage limits
- Logging and observability
- Cost tracking and optimization

**Lab:** Add per-user rate limits and token usage dashboard.

### Module 4 — Deployment
- Deploy app to cloud
- Secure secrets
- Monitor failures and regressions

**Lab:** Deploy a simple web app with health checks.

## Assessment
- End-to-end app demo
- Architecture diagram + operational runbook
