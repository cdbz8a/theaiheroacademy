# Capstone: Ship an AI App (2 weeks)

## Goal
Build and publish one useful AI application solving a real user problem.

## Required Deliverables
1. Problem statement and user persona.
2. Prompt + workflow design document.
3. Working app (hosted demo or local run instructions).
4. Evaluation report (quality, latency, cost, safety).
5. Launch retrospective and next-iteration plan.

## Recommended Project Ideas
- Meeting notes summarizer with action extraction.
- Job application tailoring assistant.
- Internal policy Q&A assistant.
- Marketing campaign brief generator.

## Grading Rubric (100 points)
- Problem clarity (15)
- Technical implementation (25)
- Output quality and evaluation rigor (25)
- Responsible AI safeguards (20)
- Demo clarity and product thinking (15)

## Demo Day Format
- 5-minute demo
- 3-minute Q&A
- Peer feedback cards
