# AI Hero Academy — Landing Page Sections

## Page Goal
Convert cold/warm traffic into enrollments or qualified calls.

## Recommended Section Order

1. **Hero + Single Offer CTA**
2. **Pain-to-Outcome Transformation**
3. **What’s Included (Program Breakdown)**
4. **Who It’s For / Not For**
5. **Learning Outcomes (Bullet Proof)**
6. **Curriculum Snapshot**
7. **Instructor Credibility**
8. **Testimonials / Case Signals**
9. **Pricing + Bonuses + Guarantee**
10. **FAQ**
11. **Final CTA + Deadline/Urgency**

---

## Copy Skeleton

## 1) Hero
**Headline:** Practical AI Skills You’ll Use Tomorrow.

**Subheadline:** A hands-on 6–8 hour workshop for professionals who want to save time, improve quality, and use AI responsibly in real-world work.

**CTA:** Enroll Now

## 2) Pain-to-Outcome
**Before:** scattered experimentation, inconsistent outputs, trust concerns  
**After:** clear method, reliable prompts, safer workflows, measurable time savings

## 3) What’s Included
- Live instruction + demos
- Guided labs with your real tasks
- Student workbook
- Prompt starter library
- Privacy + verification checklist
- 30-day rollout plan

## 4) Who It’s For / Not For
**For:** non-technical professionals, managers, founders, educators  
**Not ideal for:** advanced ML engineers seeking model-level deep dives

## 5) Learning Outcomes
- Select the right AI tool by task
- Build better prompts fast
- Improve email/meeting/document workflows
- Verify outputs and reduce risk
- Implement a practical weekly AI routine

## 6) Curriculum Snapshot
Use module highlights from canonical course files.

## 7) Credibility
- Course developed for practical workplace adoption
- Real-task workshop model (not passive lecture)
- Safety and verification integrated throughout

## 8) Testimonials
Use role-based quotes + outcome metrics where possible.

## 9) Pricing + Bonuses + Guarantee
- Main offer tiers
- Bonus resources
- Deadline and seat limits
- Guarantee language (if offered)

## 10) FAQ
- Is this beginner-friendly?
- Which tools do we use?
- Do I need paid AI subscriptions?
- Is team training available?
- Will we cover privacy/safety?

## 11) Final CTA
Repeat core value + low-friction next action.
