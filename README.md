## Quick Start

1. Open `course-materials/AI_Hero_Academy_6-8h_Course_Plan.md` to run the class.
2. Build presentation from `course-materials/AI_Hero_Academy_Slide_Deck.md` (or import outline file).
3. Print `course-materials/AI_Hero_Academy_Student_Workbook_Printable.md` as the participant packet.

## Suggested Delivery Format
- Instruction time: **~7 hours**
- Total day with breaks/lunch: **~8 to 8.5 hours**
- Audience: beginner to intermediate, non-technical professionals
